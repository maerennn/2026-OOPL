# Game Introduction
- 一款難度偏易的彈幕遊戲，玩家操控角色躲避四面八方的彈幕並通過層層關卡
- 角色: 八個方位進行移動，可切換成低速模式以進行微操，能釋放符卡消除彈幕，攻擊強度會隨著吃到的P逐漸上升，直到最大值
- 通關方式: 在生命數量未耗完之前擊破道中BOSS與關底BOSS
- 場景: 共6關，每關皆有小怪及道中BOSS、關底BOSS，每關的場景及音樂會適時進行切換
- 影片參考(Normal難度展示): https://youtu.be/8AhyKZ3ZoV0?si=2ZJVZeBRHhUUACaI
